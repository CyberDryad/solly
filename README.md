# Installation / Running the Bot

1. `npm install`
2. Set up a `config.json` with webhooks and overriden `OWNER` permissions.
3. Set your development TOKEN variable in `.env`.
4. Set your environment ENVIRONMENT variable in `.env`.

**Starting the Bot**
```
npm run start
```
